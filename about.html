AFRO GAME
DEVELOPER CONFERNECE
2022

The Afro Game Developer Conference brings together the up and coming indie game developers in Africa. This ushers a moment where game developers can network with each other and learn from one another. Major Game companies like Epic Games, Activision, Ubisoft etc will be present at this event and give talks about their latest games in the works and the kind of technologies being used.
